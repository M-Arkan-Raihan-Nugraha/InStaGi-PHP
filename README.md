### 📝 Deskripsi Proyek
**InStaGi** adalah aplikasi berbasis web yang dirancang untuk memantau status gizi pasien secara cepat dan akurat. Proyek ini dikembangkan secara khusus sebagai partisipasi dalam kegiatan **Rotinspark** dalam rangka memperingati **HUT RS Paru yang ke-91**.

Aplikasi ini berfokus pada digitalisasi monitoring dan pencatatan medis terkait status gizi menggunakan parameter standar kesehatan yang berlaku.

### ✨ Fitur Utama
* **Kalkulator IMT (Indeks Massa Tubuh):** Perhitungan otomatis berdasarkan tinggi dan berat badan pasien dengan pembulatan presisi sesuai standar medis.
* **Klasifikasi Status Gizi:** Penentuan status gizi otomatis berdasarkan standar **Kemenkes RI** (Sangat Kurus, Kurus, Normal, Gemuk, Obesitas).
* **Kebutuhan Kalori Harian:** Menghitung total kalori yang dibutuhkan pasien per hari untuk menjaga atau mencapai berat badan ideal.
* **Saran Gizi Praktis:** Memberikan rekomendasi pola makan danasupan nutrisi yang dipersonalisasi berdasarkan hasil status gizi pasien.
* **Manajemen Data Pasien:** Penyimpanan data pasien yang sistematis dan terorganisir untuk memudahkan tenaga medis dalam pemantauan berkala.
